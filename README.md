# Task-1-Data-Cleaning
Task 1 - Data Cleaning (Sales Dataset)

# Objective
Clean and preprocess a raw dataset to remove duplicates, handle missing values, fix data types, and standardize formatting.

# Steps Performed
1. Checked and removed duplicate records using Excel.
2. Identified and handled missing values:
   Numerical columns: Filled using median values
   Text columns: Filled using most frequent value
3. Standardized text:
  Trimmed extra spaces
  Converted to Proper Case format
4.Reformatted Order_Date column to standard date format (dd-mm-yyyy).
5.Recalculated Total_Amount using: =Quantity * Price
6. Renamed column headers into consistent, clean naming convention.

# Tools Used
Microsoft Excel
# Key Learning
This task helped me understand common data quality issues and practice real-world data cleaning techniques.
